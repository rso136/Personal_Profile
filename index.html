Portfolio Highlight: Essentials (MEAN version)

Update 5/30/17: When I first developed Essentials a little over a week ago, I based it on a PHP / MySQL tech stack in order to sharpen my background in PHP (check out the original app here). As my next portfolio project, I wanted to transpose Essentials from a PHP / MySQL stack to a completely different web technology stack. This time, I decided to recreate Essentials based on the MEAN (MongoDB / Express / AngularJS / Node) stack with the aim to have it function similarly as the PHP/MySQL version of Essentials.

Given how important it is to be able to transpose assets from one tech stack to another, I found that the process of recreating Essentials based on the MEAN stack really sharpened my understanding of the associated technologies.

As always, check out the git repository for a look into the inner workings of the app and most of all, try out the app itself!

The key technology stack integrated into this project is as follows:

Technology Stack

  • MongoDB
  • Express
  • AngularJS
  • Node
